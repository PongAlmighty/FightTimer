# Product Overview

FightTimer is a professional-grade browser-based countdown timer designed for live streaming and production environments. It provides real-time timer displays with customizable appearance and remote control capabilities.

## Core Features

- **Web-based timer display** - Full-screen countdown timer optimized for OBS and streaming
- **Control panel interface** - Separate control interface for timer management
- **Real-time updates** - WebSocket-based communication for instant synchronization
- **REST API integration** - Bitfocus Companion and external system support
- **Customizable appearance** - Colors, fonts, sizes, and end messages
- **Google Fonts support** - Automatic font downloading and local caching
- **Persistent settings** - Browser localStorage for configuration persistence

## Target Use Cases

- Live streaming countdown timers
- Production environment timing
- Event management and scheduling
- Remote timer control via Stream Deck/Companion
- Multi-screen timer displays

## Future Direction

The project is evolving toward multi-timer support, allowing up to 5 independent timers with separate displays, settings, and API endpoints while maintaining backward compatibility with the current single-timer implementation.
